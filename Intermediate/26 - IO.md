# IO in Go

Tags: #go #intermediate #io #reader #writer #interfaces

## Overview

The `io` package defines fundamental I/O interfaces: `Reader`, `Writer`, `Closer`, and more. Everything in Go's I/O ecosystem is built around these interfaces.

## Core Interfaces

```go
type Reader interface { Read(p []byte) (n int, err error) }
type Writer interface { Write(p []byte) (n int, err error) }
type Closer interface { Close() error }
```

## Code Examples

### Reading from a Reader
```go
func readFromReader(r io.Reader) {
    buf := make([]byte, 1024)
    n, err := r.Read(buf)
    fmt.Println(string(buf[:n]))
}
readFromReader(strings.NewReader("Hello Reader!"))
```

### Writing to a Writer
```go
var writer bytes.Buffer
writer.Write([]byte("Hello Writer!"))
fmt.Println(writer.String())
```

### bytes.Buffer
```go
var buf bytes.Buffer          // stack allocation
buf.WriteString("Hello Buffer!")
fmt.Println(buf.String())

buf2 := new(bytes.Buffer)     // heap allocation
```

### io.MultiReader
```go
r1 := strings.NewReader("Hello ")
r2 := strings.NewReader("World!")
mr := io.MultiReader(r1, r2)
buf := new(bytes.Buffer)
buf.ReadFrom(mr)
fmt.Println(buf.String())  // "Hello World!"
```

### io.Pipe (synchronous in-memory pipe)
```go
pr, pw := io.Pipe()
go func() {
    pw.Write([]byte("Hello Pipe"))
    pw.Close()
}()
buf := new(bytes.Buffer)
buf.ReadFrom(pr)
fmt.Println(buf.String())
```

### Write to file using io.Writer
```go
file, _ := os.OpenFile(filepath, os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
defer file.Close()
file.Write([]byte(data))
```

## Key Points

- `strings.NewReader` creates a `Reader` from a string.
- `bytes.Buffer` implements both `Reader` and `Writer`.
- `io.Copy(dst, src)` copies from any `Reader` to any `Writer`.
